# Graphite crucible / susceptor photos

Lab photographs of the in-house machined two-piece graphite crucible/susceptor
used in the induction furnace, contributed by @ronnie-guymon on
[PR #3](https://github.com/vertical-cloud-lab/custom-induction-furnace/pull/3#issuecomment-4847415967).

| File | Description |
| --- | --- |
| `crucible_disassembled.jpg` | Crucible disassembled: lidded cap with central pyrometer-sighting bore (left) and cup body with machined sample cavity (right), with disc specimens for scale. |
| `crucible_stacked.jpg` | The two graphite pieces nested together (oblique view), showing the stepped cap that seats into the cup body and the central sighting bore. |
| `crucible_assembled_oblique.jpg` | Crucible assembled (oblique view), with a specimen seated in the cavity and visible through the cap bore. |
| `crucible_assembled.jpg` | Crucible assembled (top view), with a specimen seated in the cavity and visible through the cap bore. |

All four are distinct photographs (`IMG_0112`, `IMG_0115`, `IMG_0117`, `IMG_0113`).

These are composed into the manuscript figure `paper/figures/fig_crucible.png`
by `paper/build_crucible_figure.py` (`make figures`).
