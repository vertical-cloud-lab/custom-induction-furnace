# Manuscript moved to LaTeX

The HardwareX manuscript is now maintained in **LaTeX** (per reviewer request — the
journal submits with Elsevier's `elsarticle` class), not Markdown.

- **Canonical manuscript:** [`paper.tex`](paper.tex)
- **Compiled PDF:** [`paper.pdf`](paper.pdf) — build with `make pdf` (pdflatex / MiKTeX).
- **Template + author guidelines:** [`template/`](template/) — mirrored `elsarticle`
  class files and a summary of the HardwareX Guide for Authors.

The previous Markdown draft of this manuscript is preserved in the repository's git
history; all of its content has been ported into `paper.tex`.
